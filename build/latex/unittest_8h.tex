A small unit test framework for C/\+C++.

Origin author\+: Bo Zhuang \href{mailto:sdzhuangbo@hotmail.com}{\texttt{ sdzhuangbo@hotmail.\+com}} Features\+:
\begin{DoxyItemize}
\item {\ttfamily ut\+\_\+assert(expr)} assertion fail if {\ttfamily expr} false
\item {\ttfamily ut\+\_\+assert(expr, message)} assertion with message
\item {\ttfamily ut\+\_\+assert(expr, message, args...)} assertion with message and args
\item {\ttfamily ut\+\_\+run\+\_\+test(test)} to run a test function in form {\ttfamily int f()} return 0 if passed
\item {\ttfamily ut\+\_\+test\+\_\+results()} to display the test results
\item {\ttfamily \#define MU\+\_\+\+NOCOLOR} if ANSI escape code not supported
\end{DoxyItemize}
